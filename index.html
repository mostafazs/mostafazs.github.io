    Jalali Calendar

    Jalali Calendar is a Firefox Add on, that show shamsi date ( Jalali Calendar ) in simply way to users. This addon show you a simple popup about today Shamsi date and a badge with today Shamsi date , it also show Persian translate of Gregorian today date. https://addons.mozilla.org/addon/jalali-persian-calendar/?src=api

  • Contribute to Jalali Calendar


    • Bits2Bytes

    Bits2Bytes is a Firefox Add on , that convert computer data units. With Bits2Bytes you can convert computer data units from bits to bytes and vice versa. This extension is perfect for computer science student, and anyone that need to convert computer units to each other. For converting , this addon use a PHP script and it need active internet connection.You can contribute on php script for make this addon better. https://addons.mozilla.org/addon/bits2bytes/?src=api

  • Contribute to Bits2Bytes
